删除分组分两种。一：只删除分组，分组下的设备会进入未分组状态。二：分组与分组下的设备一起删除

  • 正式环境URL: https://cn(区域)-api.coolkit.cn:8080/api/group

  • 测试环境URL: https://testapi.coolkit.cn:8080/api/group

  • 请求方法： DELETE

  • 请求参数：

参数名 必选 类型 说明
id 是 String 分组id
type 是 Int 0：只删除分组 1：连同分组下的设备一起删掉

响应参数:

参数名 必选 类型 说明
error 是 Int 错误码 
错误码
0:操作成功
400:参数错误
401:认证失败
500:服务器错误
